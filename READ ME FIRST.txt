====================================================================
LSR XML Helper - Important Information
====================================================================

✔ This tool allows you to view, edit and duplicate entries inside XML files
used in Los Santos RED.

✔ You MUST extract the ZIP before using it.

✔ When you run the EXE for the first time, Windows may show:
   "Windows protected your PC"

This is because the application is not commercially code-signed.

To continue:
   1. Click "More info"
   2. Click "Run anyway"

-------------------------------------------------------------------
ANTIVIRUS WARNING INFORMATION
-------------------------------------------------------------------
Some antivirus programs may show warnings or flag the file as suspicious,
unknown or potentially unsafe.

This is NOT because the tool is harmful, but because:
• It is a new EXE with no reputation yet
• It is unsigned
• It performs file editing (which AVs monitor closely)

If your antivirus blocks or deletes it, do this:
• Open your antivirus settings
• Add "LSR-XML-Helper.exe" to the allowed list / exclusions / safe list

Once added, it will run normally.

-------------------------------------------------------------------
AUTO UPDATES
-------------------------------------------------------------------
• The tool automatically checks online for newer versions.
• If an update is available, it downloads and replaces itself.
• Afterwards, simply launch the tool again.

-------------------------------------------------------------------
HOW TO USE
-------------------------------------------------------------------
When launched:
1) Select the folder containing your Los Santos RED XML files.
2) Pick an XML file to view or edit.
3) Make changes and save.

The folder is remembered automatically for next time.

-------------------------------------------------------------------
WHAT THIS TOOL DOES NOT DO
-------------------------------------------------------------------
This tool does NOT:
• Modify anything outside your chosen XML folder
• Install files onto your system
• Change registry settings
• Require admin permissions
• Send data anywhere
• Stay running after exit

It ONLY edits XML files that YOU explicitly choose.
